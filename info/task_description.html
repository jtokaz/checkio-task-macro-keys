A keyboard macro allows a sequence of keystrokes to be transformed into another usually longer sequence of keystrokes.

For this task someone has maliciously assigned macros to single letter keystrokes. If d is assigned to foo and s is assigned to bar then typing "I like dessert" results in the screen displaying "I like fooebarbarert". You know what you typed and what you see displayed on the screen. Use your coding skills to determine the minimum number of letters that were assigned macros.

Only letters are assigned macros, and these can be any length including zero. The upper and lower cases of a letter are mapped separately. The letter A could be mapped to b, while a is mapped to bar or not mapped at all.

Input: The string you typed and the string displayed with macros applied.

Output: The minimal number of macros required to transform the typed string to the displayed string.

Precondition: 0 ≤ |min_num_macros| ≤ 5
